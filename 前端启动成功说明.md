## 常见问题

### Q1: 浏览器显示空白页面？

**A**: 检查以下几点：
1. 确认后端服务是否已启动
2. 打开浏览器开发者工具（F12）查看控制台错误
3. 检查网络请求是否正常
4. 尝试清除浏览器缓存后刷新

### Q2: 端口 5173 被占用？

**A**: 修改 `vite.config.js` 中的端口：
```javascript
export default defineConfig({
  server: {
    port: 5174,  // 改为其他端口
  }
})
```

### Q3: 修改代码后页面没有更新？

**A**: Vite 支持热更新（HMR），修改代码后应该自动刷新。如果没有：
1. 检查命令行是否有错误信息
2. 手动刷新浏览器（F5）
3. 重启前端服务

### Q4: npm install 很慢？

**A**: 使用国内镜像源：
```bash
npm install --registry=https://registry.npmmirror.com
```

## 开发提示

### 热更新（HMR）
- 修改 `.vue` 文件后，页面会自动刷新
- 修改 `.js` 文件后，页面会自动刷新
- 修改 `vite.config.js` 后，需要重启服务

### 开发者工具
- 按 `F12` 打开浏览器开发者工具
- 查看 Console 标签页的错误信息
- 查看 Network 标签页的网络请求

### Vite 命令
在前端服务运行时，可以按以下快捷键：
- `h + Enter`: 显示帮助
- `r + Enter`: 重启服务
- `u + Enter`: 显示服务器 URL
- `o + Enter`: 在浏览器中打开
- `c + Enter`: 清除控制台
- `q + Enter`: 退出服务
