### 连胜竞技场测试
1. 打开浏览器开发者工具 (F12)
2. 访问连胜竞技场页面
3. 查看 Console 日志，确认数据加载正常：
   ```
   连胜竞技场信息加载成功:
     当前连胜: X
     今日最高: Y
     已领取奖励: []
   ```
4. 进行战斗，达到连胜次数
5. 点击"领取"按钮，查看是否能正常领取奖励

2. 分别挑战通天塔和龙纹塔
