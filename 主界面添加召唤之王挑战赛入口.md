# 主界面添加召唤之王挑战赛入口

## 实现时间
2026年1月18日

## 需求
把"皇城：召唤之王挑战赛"功能移到主界面最下面功能区和聊天区的中间

## 实现内容

### 1. 在主界面添加皇城区块

在主界面的"聊天区"和"导航菜单"（最下面功能区）之间添加了"皇城"区块：

**位置：**
```
【个人信息】
  等级、声望、活力、水晶塔、战力、铜钱、元宝...

【聊天区】
  聊天面板...

【皇城】          ← 新增（在聊天区下面）
  召唤之王挑战赛   ← 新增

【导航菜单】      ← 最下面功能区
  幻兽、背包、商城...
  联盟、盟战、地图...
  切磋、闯塔、战场...
  古树、排行、成就...
  兑换、签到、论坛...
```

**代码实现：**
```vue
<!-- 聊天区 -->
<div class="section title">【聊天区】</div>
<ChatPanel />

<!-- 皇城：召唤之王挑战赛 -->
<div class="section title">【皇城】</div>
<div class="section indent">
  <a class="link" @click="handleLink('召唤之王挑战赛')">召唤之王挑战赛</a>
</div>

<!-- 导航菜单 -->
```

### 2. 功能说明

- **点击效果**：点击"召唤之王挑战赛"链接会跳转到 `/king` 路由
- **样式**：使用与其他功能一致的蓝色链接样式
- **位置**：位于聊天区下方，导航菜单上方，方便玩家快速访问

### 3. 地图页面保留

地图页面（`MapPage.vue`）中的"皇城：召唤之王挑战赛"功能保持不变，因为：
1. 地图页面是游戏世界地图的展示
2. 皇城是地图的一部分，应该在地图页面显示
3. 主界面添加的是快捷入口，方便玩家直接访问

## 修改文件

**文件：** `interfaces/client/src/features/main/MainPage.vue`

**修改位置：** 在"聊天区"和"导航菜单"之间添加新区块

## 用户体验改进

### 改进前
- 玩家需要：主页 → 地图 → 找到皇城区块 → 点击召唤之王挑战赛
- 步骤较多，不够便捷

### 改进后
- 玩家可以：主页 → 向下滚动到聊天区下方 → 直接点击皇城区块的召唤之王挑战赛
- 减少操作步骤，提升访问效率
- 同时保留地图页面的入口，满足不同场景需求
