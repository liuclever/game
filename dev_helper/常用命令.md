# 常用命令速查表

## 🚀 启动相关

```bash
# 启动后端
python -m interfaces.web_api.app

# 启动前端
cd interfaces\client
npm run dev

# 构建前端生产版本
cd interfaces\client
npm run build
```

---

## 📦 依赖管理

```bash
# 安装 Python 依赖
pip install -r requirements.txt

# 安装单个 Python 包
pip install 包名

# 查看已安装的 Python 包
pip list

# 安装前端依赖
cd interfaces\client
npm install

# 安装单个 npm 包
npm install 包名

# 查看 npm 包版本
npm list
```

---

## 🗄️ 数据库操作

```bash
# 连接数据库
mysql -u root -p game_tower

# 导入 SQL 文件
mysql -u root -p game_tower < game_tower.sql

# 导出数据库
mysqldump -u root -p game_tower > backup.sql

# 查看数据库列表
mysql -u root -p -e "SHOW DATABASES;"
```

### 常用 SQL 查询

```sql
-- 查看所有表
SHOW TABLES;

-- 查看表结构
DESC players;

-- 查看玩家数据
SELECT * FROM players LIMIT 10;

-- 查看幻兽数据
SELECT * FROM player_beasts WHERE player_id = 1;

-- 创建测试玩家
INSERT INTO players (username, password, level) VALUES ('test_user', '123456', 1);

-- 清空表数据（谨慎使用）
TRUNCATE TABLE table_name;
```

## 🛠️ 项目维护

## 🐛 调试技巧

```bash
# 查看 Python 版本
python --version

# 查看 pip 版本
pip --version

# 查看 Node.js 版本
node --version

# 查看 npm 版本
npm --version

# 测试数据库连接

taskkill /F /PID 进程ID
```

## 📊 性能分析

```bash
# 查看 Python 包大小
pip list --format=freeze | wc -l

# 查看项目代码行数
git ls-files | xargs wc -l

# 查看数据库大小
mysql -u root -p -e "SELECT table_schema AS 'Database', ROUND(SUM(data_length + index_length) / 1024 / 1024, 2) AS 'Size (MB)' FROM information_schema.TABLES WHERE table_schema = 'game_tower';"
```

## 💡 快捷操作

```bash
# 一键重启后端（先停止再启动）
taskkill /F /IM python.exe
python -m interfaces.web_api.app

# 一键清理并重装前端依赖
cd interfaces\client
rmdir /s /q node_modules
npm install
cd ..\..

# 快速备份数据库
mysqldump -u root -p game_tower > backup_%date:~0,4%%date:~5,2%%date:~8,2%.sql
```
