# 完整初始化步骤

## 📋 从零开始配置项目

### 1️⃣ 检查环境

```bash
双击运行：dev_helper\环境检查.bat
```

确保所有项都显示 ✅

---

### 2️⃣ 配置数据库密码

编辑文件：`infrastructure/db/connection.py`

修改第 13 行：
```python
'password': '你的MySQL密码',  # 改成你本地的密码
```

保存文件（Ctrl + S）

---

### 3️⃣ 导入数据库

```bash
双击运行：dev_helper\导入数据库.bat
```

会提示输入 MySQL 密码，输入后等待导入完成。

---

### 4️⃣ 安装项目依赖

```bash
双击运行：dev_helper\首次安装依赖.bat
```

这一步会：
- 安装 Python 后端依赖（flask, pymysql 等）
- 安装 Node.js 前端依赖（可能需要几分钟）

---

### 5️⃣ 测试数据库连接

```bash
双击运行：dev_helper\快速测试.bat
```

如果显示 ✅ 数据库连接成功，说明配置正确！

---

### 6️⃣ 启动项目

**打开两个命令行窗口：**

**窗口 1 - 启动后端：**
```bash
双击运行：dev_helper\启动后端.bat
```

等待看到：
```
* Running on http://127.0.0.1:5000
```

**窗口 2 - 启动前端：**
```bash
双击运行：dev_helper\启动前端.bat
```

等待看到：
```
Local: http://localhost:5173
```

---

### 7️⃣ 访问项目

在浏览器打开：http://localhost:5173

现在应该可以正常使用了！

---

## 🐛 常见问题

### Q: 导入数据库时提示密码错误

**A:** 检查 MySQL 密码是否正确，或者重置 MySQL 密码

### Q: 后端启动后一直报错

**A:** 
1. 确保数据库已导入
2. 确保 `infrastructure/db/connection.py` 中的密码正确
3. 运行 `快速测试.bat` 检查连接

### Q: 前端显示 500 错误

**A:** 
1. 检查后端是否正常启动
2. 查看后端终端的错误信息
3. 确保数据库表已导入

### Q: 前端显示 ECONNREFUSED 错误

**A:** 后端没有启动，运行 `启动后端.bat`
