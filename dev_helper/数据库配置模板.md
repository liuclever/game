# 数据库配置模板

## 📝 配置文件位置
`infrastructure/db/connection.py`

## 🔧 配置内容

```python
"""
MySQL数据库连接配置
"""
import pymysql
from pymysql.cursors import DictCursor


# 数据库配置
DB_CONFIG = {
    'host': 'localhost',        # 数据库地址（本地开发用 localhost）
    'port': 3306,               # MySQL 端口（默认 3306）
    'user': 'root',             # 数据库用户名
    'password': '你的密码',      # ⚠️ 改成你本地的 MySQL 密码
    'database': 'game_tower',   # 数据库名称
    'charset': 'utf8mb4',       # 字符集
    'cursorclass': DictCursor,  # 返回字典格式
}


def get_connection():
    """获取数据库连接"""
    return pymysql.connect(**DB_CONFIG)


def execute_query(sql: str, params: tuple = None) -> list:
    """执行查询，返回结果列表"""
    conn = get_connection()
    try:
        with conn.cursor() as cursor:
            cursor.execute(sql, params)
            return cursor.fetchall()
    finally:
        conn.close()


def execute_update(sql: str, params: tuple = None) -> int:
    """执行更新，返回影响行数"""
    conn = get_connection()
    try:
        with conn.cursor() as cursor:
            result = cursor.execute(sql, params)
            conn.commit()
            return result
    finally:
        conn.close()


def execute_insert(sql: str, params: tuple = None) -> int:
    """执行插入，返回自增ID"""
    conn = get_connection()
    try:
        with conn.cursor() as cursor:
            cursor.execute(sql, params)
            conn.commit()
            return cursor.lastrowid
    finally:
        conn.close()
```

## 🗄️ 数据库初始化步骤

### 1. 创建数据库
```sql
CREATE DATABASE game_tower CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. 导入数据库结构
```bash
mysql -u root -p game_tower < game_tower.sql
```

### 3. 验证数据库
```sql
USE game_tower;
SHOW TABLES;
```

## ⚠️ 注意事项

1. **密码安全**：不要将真实密码提交到 Git 仓库
2. **字符集**：必须使用 utf8mb4 支持中文和 emoji
3. **连接池**：生产环境建议使用连接池（如 DBUtils）
4. **环境变量**：建议使用环境变量管理敏感配置

## 🔐 使用环境变量（推荐）

创建 `.env` 文件（不要提交到 Git）：
```
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=game_tower
```

修改 `connection.py`：
```python
import os
from dotenv import load_dotenv

load_dotenv()

DB_CONFIG = {
    'host': os.getenv('DB_HOST', 'localhost'),
    'port': int(os.getenv('DB_PORT', 3306)),
    'user': os.getenv('DB_USER', 'root'),
    'password': os.getenv('DB_PASSWORD'),
    'database': os.getenv('DB_NAME', 'game_tower'),
    'charset': 'utf8mb4',
    'cursorclass': DictCursor,
}
```

安装依赖：
```bash
pip install python-dotenv
```
