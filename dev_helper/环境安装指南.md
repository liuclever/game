# 环境安装指南

## 🔍 第一步：检查环境

双击运行 `环境检查.bat`，查看哪些软件需要安装。

---

## 🐍 安装 Python 3.10+

### 下载 Python

访问官网：https://www.python.org/downloads/

推荐下载：**Python 3.10** 或 **Python 3.11**

### 安装步骤

1. **运行安装程序**

2. **⚠️ 重要：勾选 "Add Python to PATH"**
   ```
   ☑ Add python.exe to PATH  ← 必须勾选！
   ```

3. **选择 "Install Now"** 或 "Customize installation"

4. **等待安装完成**

5. **验证安装**
   ```bash
   # 打开新的命令行窗口
   python --version
   pip --version
   ```

### 如果忘记勾选 PATH

**方法一：重新安装**
- 卸载 Python
- 重新安装，记得勾选 "Add Python to PATH"

**方法二：手动添加环境变量**
1. 找到 Python 安装路径（通常是 `C:\Users\你的用户名\AppData\Local\Programs\Python\Python310\`）
2. 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
3. 在"系统变量"中找到 `Path`，点击"编辑"
4. 添加两个路径：
   - `C:\Users\你的用户名\AppData\Local\Programs\Python\Python310\`
   - `C:\Users\你的用户名\AppData\Local\Programs\Python\Python310\Scripts\`
5. 点击"确定"保存
6. **重启命令行窗口**

## 📦 安装 Node.js 18+

### 下载 Node.js

访问官网：https://nodejs.org/

推荐下载：**LTS 版本**（长期支持版）

### 安装步骤

1. **运行安装程序**

2. **一路点击 "Next"**（默认会自动添加到 PATH）

3. **等待安装完成**

4. **验证安装**
   ```bash
   # 打开新的命令行窗口
   node --version
   npm --version
   ```

### 配置 npm 镜像（可选，加速下载）

```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com
```

## 🗄️ 安装 MySQL 8.0

### 下载 MySQL

访问官网：https://dev.mysql.com/downloads/mysql/

选择：**MySQL Community Server 8.0**

### 安装步骤

1. **运行安装程序**

2. **选择安装类型**
   - 推荐：Developer Default（开发者默认）

3. **配置 MySQL Server**
   - Authentication Method：使用默认的强密码加密
   - Root Password：设置 root 用户密码（**记住这个密码！**）

4. **完成安装**

5. **验证安装**
   ```bash
   # 打开新的命令行窗口
   mysql --version
   ```

### 如果 mysql 命令不可用

手动添加到环境变量：
1. 找到 MySQL 安装路径（通常是 `C:\Program Files\MySQL\MySQL Server 8.0\bin\`）
2. 添加到系统环境变量 `Path` 中
3. 重启命令行窗口

## 🎯 安装完成后的步骤

### 1. 创建数据库

```bash
# 连接 MySQL
mysql -u root -p

# 输入密码后，执行以下 SQL
CREATE DATABASE game_tower CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 退出
exit;
```

### 2. 导入数据库结构

```bash
mysql -u root -p game_tower < game_tower.sql
```

### 3. 修改数据库配置

编辑 `infrastructure/db/connection.py`：

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',
    'password': '你刚才设置的MySQL密码',  # ← 改这里
    'database': 'game_tower',
    'charset': 'utf8mb4',
    'cursorclass': DictCursor,
}
```

### 4. 安装项目依赖

```bash
# 运行安装脚本
dev_helper\首次安装依赖.bat
```

### 5. 测试连接

```bash
# 运行测试脚本
dev_helper\快速测试.bat
```

### 6. 启动项目

```bash
# 启动后端
dev_helper\启动后端.bat

# 启动前端（新开一个命令行窗口）
dev_helper\启动前端.bat
```

## 🐛 常见问题

### Q: 安装 Python 后，命令行还是提示找不到 python

**A:** 需要重启命令行窗口，或者重启电脑

### Q: pip 安装很慢

**A:** 使用国内镜像源
```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: npm install 很慢

**A:** 使用淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: MySQL 安装失败

**A:** 
1. 确保之前没有安装过 MySQL（卸载旧版本）
2. 以管理员身份运行安装程序
3. 关闭杀毒软件

### Q: 忘记 MySQL root 密码

**A:** 
1. 停止 MySQL 服务
2. 以安全模式启动 MySQL
3. 重置密码
4. 详细步骤请搜索"MySQL 重置 root 密码"
