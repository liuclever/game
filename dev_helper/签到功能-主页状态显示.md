# 签到功能 - 主页状态显示

## 更新内容

### 主页签到状态显示
- **未签到**：显示"今日 签到"（蓝色链接，可点击跳转到签到页面）
- **已签到**：显示"今日 已签到"（黑色加粗文字，不可点击）

### 实现细节

1. 添加了 `signinStatus` 响应式对象，包含：
   - `hasSigned`: 是否已签到
   - `loading`: 加载状态

2. 添加了 `loadSigninStatus()` 方法：
   - 调用 `/api/signin/info` 接口获取签到状态
   - 在用户登录后自动调用

3. 修改了签到链接的显示逻辑：
   - 使用 `v-if` 条件渲染
   - 已签到时显示加粗文字
   - 未签到时显示可点击链接

## 测试步骤

1. 刷新主页（Ctrl+F5）
2. 登录账号
3. 查看"今日"后面的签到状态
4. 如果未签到，点击"签到"链接进入签到页面
5. 签到成功后，返回主页应该显示"已签到"

## 相关文件

- `interfaces/client/src/features/main/MainPage.vue` - 主页组件
- `interfaces/web_api/app.py` - 签到API接口
- `application/services/signin_service.py` - 签到服务
