# 切磋功能说明

## 功能概述

切磋是玩家之间的友谊赛，不消耗活力，不影响排名，但会记录战绩。

## 使用流程

1. **进入玩家主页**
   - 在任何页面点击玩家名字
   - 跳转到玩家详情页面（`/player/detail?id=xxx`）

2. **发起切磋**
   - 在玩家主页的"战绩"旁边有"切磋"按钮
   - 点击"切磋"按钮发起战斗

3. **查看结果**
   - 瞬间显示切磋结果页面
   - 包含：
     - 凝神香加成
     - 对战双方名字
     - 幻兽经验（切磋不给经验）
     - 战斗结果（完美胜利/胜利/失败）
     - 活力消耗（显示-15，实际不消耗）
     - 战斗过程摘要
     - 切磋战绩统计

## 数据库表

### spar_records（切磋战绩表）

```sql
CREATE TABLE spar_records (
    id INT AUTO_INCREMENT PRIMARY KEY,
    attacker_id INT NOT NULL,
    defender_id INT NOT NULL,
    is_victory TINYINT(1) NOT NULL DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 前端页面

### 玩家详情页
- 路径：`interfaces/client/src/features/player/PlayerDetailPage.vue`
- 显示玩家信息和"切磋"按钮

### 切磋战报页
- 路径：`interfaces/client/src/features/player/SparBattleReportPage.vue`
- 显示切磋结果和战斗过程

## 部署步骤

1. 运行部署脚本：
   ```bash
   cd dev_helper
   部署切磋功能.bat
   ```

2. 重启后端服务

3. 测试功能：
   - 注册两个账号
   - 用账号A点击账号B的名字进入主页
   - 点击"切磋"按钮
   - 查看切磋结果

## 注意事项

- 切磋不消耗活力（虽然显示-15）
- 切磋不给幻兽经验
- 切磋会记录战绩
- 不能与自己切磋
- 双方都需要有出战幻兽
