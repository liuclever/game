# 手动安装依赖指南

如果自动安装脚本失败，可以按照以下步骤手动安装。

---

## 📋 前提条件

确保已经安装：
- Python 3.10+
- Node.js 18+
- MySQL 8.0

运行 `环境检查.bat` 确认环境正常。

---

## 🐍 手动安装 Python 依赖

### 1. 打开命令行

按 `Win + R`，输入 `cmd`，回车

### 2. 进入项目目录

```bash
cd /d D:\sxsm\project\sxsm_game\game-dev
```

### 3. 安装依赖

```bash
pip install flask
pip install flask-cors
pip install pymysql
pip install APScheduler
pip install pyngrok
```

或者一次性安装：

```bash
pip install flask flask-cors pymysql APScheduler pyngrok
```

### 4. 验证安装

```bash
pip list
```

应该能看到刚才安装的包。

---

## 📦 手动安装前端依赖

### 1. 进入前端目录

```bash
cd interfaces\client
```

### 2. 安装依赖

```bash
npm install
```

这个过程可能需要几分钟，请耐心等待。

### 3. 返回项目根目录

```bash
cd ..\..
```

---

## ✅ 验证安装

### 测试 Python 依赖

```bash
python -c "import flask; print('Flask 安装成功')"
python -c "import pymysql; print('PyMySQL 安装成功')"
```

### 测试前端依赖

```bash
cd interfaces\client
dir node_modules
cd ..\..
```

应该能看到 `node_modules` 文件夹。

---

## 🚀 安装完成后

运行测试脚本：

```bash
dev_helper\快速测试.bat
```

如果显示 "✅ 数据库连接成功！"，说明环境配置完成！

---

## 🐛 常见错误

### 错误 1: pip 安装时提示权限不足

**解决方案：**
```bash
# 使用 --user 参数
pip install --user flask flask-cors pymysql APScheduler pyngrok
```

### 错误 2: pip 安装超时

**解决方案：**
```bash
# 使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple flask flask-cors pymysql APScheduler pyngrok
```

### 错误 3: npm install 失败

**解决方案：**
```bash
# 清理缓存
npm cache clean --force

# 删除 node_modules 和 package-lock.json
cd interfaces\client
rmdir /s /q node_modules
del package-lock.json

# 重新安装
npm install
```

### 错误 4: npm install 提示网络错误

**解决方案：**
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
npm install
```
