## 🎯 快速开始

### 第一次运行项目

1. **检查环境**
   ```bash
   双击运行：环境检查.bat
   ```
   如果有 ❌ 标记，请先安装对应软件（参考 `环境安装指南.md`）

2. **安装依赖**
   ```bash
   双击运行：首次安装依赖.bat
   ```
   如果失败，参考 `手动安装依赖.md`

3. **配置数据库**
   - 参考 `数据库配置模板.md`
   - 修改 `infrastructure/db/connection.py` 中的密码

4. **测试连接**
   ```bash
   双击运行：快速测试.bat
   ```

5. **启动服务**
   ```bash
   双击运行：启动后端.bat
   双击运行：启动前端.bat
   ```

6. **访问项目**
   - 前端：http://localhost:5173
   - 后端：http://127.0.0.1:5000

## 💡 使用建议

### 日常开发流程

1. 打开两个终端窗口
2. 终端1：运行 `启动后端.bat`
3. 终端2：运行 `启动前端.bat`
4. 开始开发，修改代码会自动热更新

### 遇到问题时

1. 查看终端的错误日志
2. 运行 `快速测试.bat` 检查数据库连接
3. 查看 `本地调试指南.md` 的常见问题部分
4. 在 `开发笔记.md` 中记录问题和解决方案

### 学习项目时

1. 先看 `项目架构图.md` 了解整体结构
2. 阅读 `../BookForAi/` 目录下的文档
3. 查看 `../configs/` 目录了解游戏配置
4. 运行 `../tests/` 目录下的测试用例
