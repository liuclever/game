## API 接口

### 获取签到信息
```
GET /api/signin/info
```

返回：
```json
{
  "ok": true,
  "hasSigned": false,
  "consecutiveDays": 0,
  "currentMonth": "1",
  "currentYear": 2026,
  "signinDays": []
}
```

### 执行签到
```
POST /api/signin
```

返回：
```json
{
  "ok": true,
  "message": "签到成功！获得铜钱：5024（连续签到1天）\n颁发者：系统",
  "gold_reward": 5024,
  "base_gold": 5024,
  "consecutive_days": 1,
  "is_doubled": false,
  "sponsor": "系统",
  "user": { ... }
}
```
