# 竞技主页 - 显示可切磋对手

## 修改内容

### 问题
竞技主页（ArenaIndex）中连胜竞技场下方显示的是固定的文字，而不是真实的可切磋对手。

### 解决方案
修改为从连胜竞技场接口获取对手列表，显示可以切磋的对手（和连胜竞技场内部一样）。

## 实现细节

### 1. 使用连胜竞技场接口

**接口**：`GET /api/arena-streak/info`

**功能**：获取连胜竞技场信息，包括对手列表

**返回数据**：
```json
{
  "ok": true,
  "current_streak": 0,
  "max_streak_today": 0,
  "opponents": [
    {
      "user_id": 4053,
      "nickname": "玩家A",
      "level": 70
    },
    {
      "user_id": 4054,
      "nickname": "玩家B",
      "level": 70
    }
  ],
  "refresh_seconds": 300,
  "streak_king": {...},
  "claimed_rewards": []
}
```

### 2. 前端修改

**文件**：`interfaces/client/src/views/ArenaIndex.vue`

**修改点**：
1. 使用 `/api/arena-streak/info` 接口获取对手列表
2. 只显示前2个对手
3. 点击玩家名字：跳转到玩家详情页
4. 点击"切磋"：跳转到连胜竞技场并自动发起战斗

**显示格式**：
```
连胜竞技场:连胜50
  玩家A (70级). 切磋
  玩家B (70级). 切磋
```

## 对手匹配规则

对手列表由连胜竞技场系统提供，匹配规则：
- 随机匹配在线玩家
- 等级相近的玩家
- 每5分钟自动刷新
- 可消耗元宝立即刷新

## 交互流程

1. **查看对手**
   - 页面加载时自动获取对手列表
   - 显示前2个可切磋的对手

2. **点击玩家名字**
   - 跳转到玩家详情页
   - 可以查看对方的详细信息

3. **点击"切磋"**
   - 跳转到连胜竞技场页面
   - 携带对手ID参数
   - 自动发起战斗

## 与连胜竞技场的关系

### 竞技主页（ArenaIndex）
- 显示前2个对手
- 快速入口，方便玩家快速切磋
- 点击"切磋"跳转到连胜竞技场

### 连胜竞技场（ArenaStreak）
- 显示完整的对手列表
- 显示连胜信息、奖励等
- 实际进行战斗的地方

## 测试步骤

1. 访问竞技主页：`/arena/index`
2. 查看连胜竞技场下方是否显示真实的对手
3. 点击玩家名字，应该跳转到玩家详情页
4. 点击"切磋"，应该跳转到连胜竞技场
5. 如果没有对手，显示"暂无对手"

## 相关文件

- `interfaces/routes/arena_streak_routes.py` - 连胜竞技场后端接口
- `interfaces/client/src/views/ArenaIndex.vue` - 竞技主页
- `interfaces/client/src/views/ArenaStreak.vue` - 连胜竞技场

## 注意事项

1. 对手列表来自连胜竞技场系统
2. 竞技主页只显示前2个对手
3. 点击"切磋"会跳转到连胜竞技场进行战斗
4. 如果连胜竞技场未开放，可能无法获取对手
