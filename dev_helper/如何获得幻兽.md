# 如何获得幻兽

## 🎮 游戏中获得幻兽的方式

### 1. 使用召唤球（推荐新手）

**步骤**：
1. 进入**背包**（主页点击"背包"链接）
2. 找到**召唤球**类道具（如"青龙召唤球"、"朱雀召唤球"等）
3. 点击召唤球，选择**"使用"**
4. 确认后即可获得对应的幻兽

**召唤球来源**：
- 新手礼包
- 商城购买
- 任务奖励
- 活动赠送

**召唤球ID范围**：20001-30000

---

### 2. 副本捕捉

**步骤**：
1. 进入**副本**（主页选择副本，如"森林入口"、"宁静之森"等）
2. 在副本中**遇到幻兽**
3. 使用**捕捉球**尝试捕捉
4. 捕捉成功后幻兽加入队伍

**捕捉成功率**：
- 取决于捕捉球类型（普通/强力/超级）
- 取决于幻兽品质和等级
- 可以多次尝试

**相关文件**：
- `interfaces/routes/dungeon_routes.py` - `/capture` 接口
- `application/services/capture_service.py` - 捕捉逻辑

---

### 3. 地图探索捕捉

**步骤**：
1. 在**地图**上移动
2. 随机遇到野生幻兽
3. 使用捕捉球捕捉

**注意**：
- 不同地图有不同的幻兽
- 幻兽出现概率不同

**相关配置**：
- `configs/map_beasts.json` - 地图幻兽配置
- `configs/map_dungeons.json` - 地图副本配置

---

### 4. 镇妖塔奖励

**步骤**：
1. 进入**镇妖塔**
2. 挑战守关幻兽
3. 胜利后有概率获得幻兽

**相关文件**：
- `interfaces/routes/zhenyao_routes.py` - 镇妖塔路由

---

### 5. 活动奖励

**来源**：
- 签到奖励
- 任务完成奖励
- 排行榜奖励
- 联盟奖励
- 节日活动

## 🔧 开发者测试：快速获得幻兽

### 方法3：使用后端API（如果有管理接口）

如果有管理员接口，可以直接调用：
```python
# 在 Python 控制台或脚本中
from interfaces.web_api.bootstrap import services

user_id = 1
template_id = 1  # 青龙
realm = 1  # 凡界
level = 1

beast = services.beast_service.obtain_beast_randomly(
    user_id=user_id,
    template_id=template_id,
    realm=realm,
    level=level
)
print(f"获得幻兽: {beast.name}")
```

## ⚠️ 注意事项

### 1. 出战幻兽设置
获得幻兽后，需要设置为**出战状态**才能参加战斗：
1. 进入**幻兽管理**
2. 选择幻兽
3. 点击**"出战"**按钮

### 2. 幻兽数量限制
- 背包有幻兽数量上限
- 出战幻兽最多4只
- 可以通过升级扩展背包

### 3. 幻兽品质
- 普通（白色）
- 精英（绿色）
- 稀有（蓝色）
- 史诗（紫色）
- 传说（橙色）
- 神兽（红色）

品质越高，属性越好。

## 🐛 常见问题

### Q: 为什么点击切磋没反应？
**A**: 可能是因为没有出战幻兽。请确保：
1. 你有至少1只幻兽
2. 幻兽已设置为出战状态
3. 对手也有出战幻兽

### Q: 如何查看我的幻兽？
**A**: 主页点击"幻兽管理"或"背包"查看。

### Q: 召唤球在哪里获得？
**A**: 
- 新手可能没有召唤球
- 需要通过任务、商城、活动获得
- 开发测试可以直接在数据库添加

### Q: 如何快速测试战斗系统？
**A**: 
1. 在数据库给自己添加幻兽（方法2）
2. 设置为出战状态
3. 进入连胜竞技场或擂台测试

## 📚 相关文件

### 后端
- `application/services/beast_service.py` - 幻兽服务
- `application/services/inventory_service.py` - 背包服务（使用召唤球）
- `application/services/capture_service.py` - 捕捉服务
- `interfaces/routes/beast_routes.py` - 幻兽路由
- `interfaces/routes/dungeon_routes.py` - 副本路由（捕捉）

### 配置
- `configs/beast_templates.json` - 幻兽模板配置
- `configs/items.json` - 道具配置（召唤球）
- `configs/map_beasts.json` - 地图幻兽配置

### 数据库
- `player_beast` 表 - 玩家幻兽数据
- `inventory` 表 - 玩家背包数据
