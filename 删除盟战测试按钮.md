# 删除盟战测试按钮说明

## 文件位置

测试按钮组件文件：
- `interfaces/client/src/features/alliance/AllianceWarTestButton.vue`

## 删除步骤

### 方法1：完全删除（推荐）

1. **删除测试按钮组件文件**
   ```bash
   rm interfaces/client/src/features/alliance/AllianceWarTestButton.vue
   ```

2. **修改 `AllianceWarPage.vue`**
   
   打开文件：`interfaces/client/src/features/alliance/AllianceWarPage.vue`
   
   删除以下内容：
   
   a) 在 `<script setup>` 部分，删除导入语句：
   ```javascript
   // 测试按钮组件（可删除）
   import AllianceWarTestButton from './AllianceWarTestButton.vue'
   ```
   
   b) 在 `<template>` 部分，删除测试按钮组件：
   ```vue
   <!-- 测试按钮组件（可删除） -->
   <AllianceWarTestButton />
   ```

### 方法2：仅隐藏（不删除文件）

如果只是想临时隐藏测试按钮，可以在 `AllianceWarPage.vue` 中注释掉相关代码：

```vue
<!-- 
<AllianceWarTestButton />
-->
```

## 验证删除

删除后，刷新盟战页面，确认测试按钮区域不再显示。

## 注意事项

- 删除前请确保不再需要测试功能
- 测试按钮会跳过时间检查，直接执行对战流程
- 建议在生产环境中删除此功能
