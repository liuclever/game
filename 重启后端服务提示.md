# 重启后端服务提示

## 修复内容

已修复地图副本法攻型幻兽的法攻值为0的问题。

## 需要重启后端服务

修改了以下文件：
- `interfaces/routes/dungeon_routes.py` - 修复法攻值读取逻辑

**请重启后端服务使修改生效**。

## 重启方法

### 方法1：使用启动脚本
```bash
start-backend.bat
```

### 方法2：手动启动
```bash
cd D:\sxsm\game-dev
python -m interfaces.web_api.app
```

## 验证修复

重启后端服务后，可以通过以下方式验证修复：

1. 进入地图副本功能
2. 挑战包含法攻型幻兽的副本（如"天罚山"、"幻灵湖畔"等）
3. 观察战斗过程中法攻型幻兽的伤害输出
4. 验证法攻型幻兽能够造成正常的法术伤害
