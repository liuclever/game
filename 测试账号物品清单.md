## 如果看不到物品

1. **刷新页面**: 按 F5 或 Ctrl+F5
2. **重新登录**: 退出后重新登录
3. **清除缓存**: 清除浏览器缓存后重新登录
4. **检查背包**: 确认在"背包"而不是"临时背包"中查看

## 技能书口袋使用方法

1. 打开背包
2. 找到"技能书口袋"
3. 点击"打开"或"使用"按钮
4. 随机获得53种技能书中的一本

## 灵力水晶使用方法

1. 打开背包
2. 找到"灵力水晶"
3. 点击"使用"按钮
4. 跳转到战灵页面兑换灵力

## 镇妖宝箱使用方法

1. 打开背包
2. 找到"五星试炼宝箱"或"五星炼狱宝箱"
3. 点击"打开"按钮
4. 获得铜钱、活力草、结晶等奖励
