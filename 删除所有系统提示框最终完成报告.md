## 修改方式

### useMessage/showMessage
- 删除 `import { useMessage } from '@/composables/useMessage'`
- 删除 `const { message, messageType, showMessage } = useMessage()`
- 删除模板中的 `<div v-if="message" class="message">`
- 删除样式中的message相关CSS
- 将 `showMessage()` 替换为 `console.log()` 或 `console.error()`

### useToast/toast
- 删除 `import { useToast } from '@/composables/useToast'`
- 删除 `const { toast } = useToast()`
- 将 `toast.error()` 替换为 `console.error()`

### alert弹窗
- 将所有 `alert()` 替换为 `console.error()`
