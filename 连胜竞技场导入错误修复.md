# 连胜竞技场导入错误修复

## 问题描述

用户尝试领取连胜奖励时，前端显示错误：
```
cannot import name 'ItemRepoFromConfig' from 'infrastructure.config.item_repo_from_config'
```

## 问题原因

在 `interfaces/routes/arena_streak_routes.py` 文件的第442行，代码尝试导入 `ItemRepoFromConfig` 类：

```python
from infrastructure.config.item_repo_from_config import ItemRepoFromConfig
```

但是实际的类名是 `ConfigItemRepo`，不是 `ItemRepoFromConfig`。

## 解决方案

修改导入语句，使用正确的类名：

**修改前**：
```python
from infrastructure.config.item_repo_from_config import ItemRepoFromConfig
item_repo = ItemRepoFromConfig()
```

**修改后**：
```python
from infrastructure.config.item_repo_from_config import ConfigItemRepo
item_repo = ConfigItemRepo()
```

## 修改文件

- `interfaces/routes/arena_streak_routes.py` (第442-443行)

## 测试验证

1. 重启后端服务
2. 进入连胜竞技场
3. 达到连胜次数
4. 点击"领取"按钮
5. 确认能够成功领取奖励
