## 优化效果

### 修改前的问题
- ❌ "七类结晶×3" 不够直观
- ❌ 玩家不知道具体获得了哪些结晶
- ❌ 需要打开背包才能查看

### 修改后的优点
- ✓ 直接显示具体的结晶名称
- ✓ 一目了然，清晰明确
- ✓ 提升用户体验

## 随机性说明

每次打开宝箱，获得的结晶种类是随机的：

**示例1：** 打开三星试炼宝箱
```
第1次：金之结晶×1、木之结晶×1、水之结晶×1
第2次：火之结晶×1、土之结晶×1、风之结晶×1
第3次：金之结晶×1、电之结晶×1、土之结晶×1
```

**示例2：** 打开七星炼狱宝箱
```
每次都会获得全部7种结晶（顺序可能不同）
```
