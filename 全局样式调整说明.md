## 四、技术细节

### 4.1 优先级策略
使用 `!important` 确保全局样式优先级最高，覆盖所有组件内部的字体设置。

### 4.2 选择器覆盖
```css
/* 覆盖所有页面容器 */
[class*="page"], 
[class*="Page"],
.container, 
.wrapper,
.section,
.content {
  font-size: 18px !important;
  background-color: #FFFFFF !important;
}
```

### 4.3 特定元素
针对常用的小字体元素进行统一增大：
- `.small` 类: 16px
- `.message-input`: 18px
- `.page-input`: 18px
- `.btn`: 18px
- 所有按钮和输入框: 18px
