# 连胜竞技场奖励调整说明

## 修改内容

### 6连胜奖励调整
**神·逆鳞碎片数量：** 1个 → 5个

## 修改前后对比

### 修改前
- 铜钱：150,000
- 重生丹：2个
- 神·逆鳞碎片：**1个**

### 修改后
- 铜钱：150,000
- 重生丹：2个
- 神·逆鳞碎片：**5个** ⭐

## 修改理由

1. **提升奖励价值** - 神·逆鳞碎片是稀有材料，需要100个才能合成1个神·逆鳞
2. **鼓励达成6连胜** - 更丰厚的奖励激励玩家达成6连胜
3. **平衡游戏进度** - 5个碎片相比1个更有意义，但仍需多次达成才能合成

## 完整连胜奖励一览

| 连胜次数 | 铜钱 | 道具奖励 |
|---------|------|---------|
| 1连胜 | 1,000 | 双倍卡×1 + 随机结晶×1 |
| 2连胜 | 5,000 | 强力捕捉球×1 + 随机结晶×1 |
| 3连胜 | 10,000 | 化仙丹×1 + 随机结晶×1 |
| 4连胜 | 50,000 | 活力草×1 + 随机结晶×1 |
| 5连胜 | 100,000 | 活力草×2 + 小喇叭×2 |
| 6连胜 | 150,000 | 重生丹×2 + 神·逆鳞碎片×5 ⭐ |
